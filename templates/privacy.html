Kebijakan Privasi Web Toolkit

Terakhir diperbarui: 18 November 2025

Web Toolkit menghargai privasi pengunjung. Ringkasan kebijakan inti:

  1. File: File yang kamu unggah diproses untuk tujuan yang kamu minta. Untuk tools client-side file tidak dikirim ke server. Untuk tools server-side, file diproses sementara dan tidak disimpan secara permanen oleh sistem.
  2. Logging: Kami menyimpan log teknis (error, penggunaan) tanpa mengaitkannya ke identitas personal.
  3. Analitik: Menggunakan Google Tag Manager untuk mengumpulkan metrik penggunaan (tanpa PII).
  4. Penyimpanan: Tidak ada penyimpanan jangka panjang file pengguna kecuali bila ada fitur eksplisit yang meminta penyimpanan.
  5. Permintaan pengguna: Kamu dapat menghubungi kami untuk meminta penghapusan data atau penjelasan lebih lanjut.
