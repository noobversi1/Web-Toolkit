Tentang Web Toolkit

Web Toolkit — Alat Ringkas untuk Hal yang Sering Dibutuhkan

Koleksi alat web kecil, cepat, dan fungsional untuk pekerjaan sehari-hari: kompres, konversi, ringkasan, pengamanan password, dan lainnya — semua dibuat supaya gampang dipakai tanpa drama.

Web Toolkit lahir karena hal-hal simpel sering merepotkan. Daripada install aplikasi, buka CLI, atau bolak-balik kirim file ke banyak layanan — di sini kami sediakan beberapa utilitas yang berjalan langsung di browser atau di server ringan yang kami kelola.

Prinsip desain

  • Ringan & efisien: fitur fokus, bukan fitur yang menumpuk.
  • Privasi-aware: alat client-side tidak mengirim file ke server; alat server-side memproses file sementara dan tidak menyimpan file.
  • Bebas ribet: UI simpel, langsung paham, hasil cepat.

Tool utama

Beberapa tool yang tersedia: Kompres Gambar, Konversi Gambar, Perbesar Gambar (upscale), Tulis Ulang (paraphraser), Pembuat Kata Sandi, Summarizer, dan lainnya. Setiap tool mencantumkan apakah prosesnya terjadi di perangkatmu atau di server.

Kontribusi & Lisensi

Proyek ini dibuat agar mudah dikembangkan. Jika kamu mau kontribusi, buka repository atau kirim isu ke kontak. Lisensi default: MIT.

Siapa yang membuat?

Web Toolkit dikembangkan dan dipelihara oleh tim kecil / pengembang independen. Jika kamu ingin dukungan berbayar, integrasi self-host, atau layanan enterprise, tidak ada di Web Toolkit ini.

Kalau mau fitur baru, jangan malu-malu. Request aja — kita prioritaskan yang sering dipakai.
